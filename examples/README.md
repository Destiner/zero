# Examples

These are standalone HTML apps generated with Zero and intended to be served
directly from GitHub Pages.

- [Tic-Tac-Toe](./tic-tac-toe.html)
- [Conway's Game of Life](./conway.html)
- [Pathfinding Visualizer](./pathfinding-visualizer.html)
- [Pixel Editor](./pixel-editor.html)
- [Local Kanban Board](./kanban-board.html)

## Prompts

### Tic-Tac-Toe

```txt
Create or overwrite examples/tic-tac-toe.html as a polished standalone browser game.

Requirements:
- Single HTML file with embedded CSS and JavaScript.
- No external dependencies, fonts, images, CDNs, build tools, or network calls.
- Human plays X against a simple computer O.
- Include score tracking, draw detection, win highlighting, reset controls, keyboard/mouse support, and responsive layout.
- Make it look launch-quality on desktop and mobile.
- Modify only examples/tic-tac-toe.html.
```

### Conway's Game of Life

```txt
Create or overwrite examples/conway.html as a polished standalone Conway's Game of Life app.

Requirements:
- Single HTML file with embedded CSS and JavaScript.
- No external dependencies, fonts, images, CDNs, build tools, or network calls.
- Include play/pause, step, clear, randomize, speed control, generation/live-cell stats, drawing on the grid, and at least three named pattern presets.
- Make it responsive and usable on desktop and mobile.
- Modify only examples/conway.html.
```

### Pathfinding Visualizer

```txt
Create or overwrite examples/pathfinding-visualizer.html as a polished standalone pathfinding visualizer.

Requirements:
- Single HTML file with embedded CSS and JavaScript.
- No external dependencies, fonts, images, CDNs, build tools, or network calls.
- Let users draw walls, move start and goal nodes, generate a random maze, clear the board, and run both A* and breadth-first search.
- Animate visited cells and the final path. Show status, visited count, and path length.
- Make it responsive and usable on desktop and mobile.
- Modify only examples/pathfinding-visualizer.html.
```

### Pixel Editor

```txt
Create or overwrite examples/pixel-editor.html as a polished standalone pixel editor.

Requirements:
- Single HTML file with embedded CSS and JavaScript.
- No external dependencies, fonts, images, CDNs, build tools, or network calls.
- Include a 32x32 grid, color picker, palette, paint/erase/fill tools, clear button, live preview, and JSON export/import textarea.
- Make drawing work with pointer input and keep the layout usable on desktop and mobile.
- Modify only examples/pixel-editor.html.
```

### Local Kanban Board

```txt
Create or overwrite examples/kanban-board.html as a polished standalone local Kanban board.

Requirements:
- Single HTML file with embedded CSS and JavaScript.
- No external dependencies, fonts, images, CDNs, build tools, or network calls.
- Include Todo, Doing, and Done columns, task creation, priority selection, delete controls, drag-and-drop between columns, and localStorage persistence.
- Seed the board with a few launch-prep themed tasks.
- Make it responsive and usable on desktop and mobile.
- Modify only examples/kanban-board.html.
```
